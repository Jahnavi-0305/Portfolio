Project 1

Photo Sharing Application

I developed a full-stack social media application as part of my Software System Design and Implementation course, using the MERN stack (MongoDB, Express.js, React, Node.js). The app features a real-time activity feed, user mentions, and engagement metrics such as likes and comments. It also includes a dynamic sidebar to track user activity. I implemented CRUD operations for comments and photos, and built user management with role-based access control. Utilizing MongoDB as a NoSQL database ensured scalable data storage with high availability and low-latency queries.
Tools Used: MongoDB, Express.js, React, Node.js (MERN Stack).

Project 2

Personal Nutrition Recommendation System

I developed a machine learning-based recommendation engine that personalizes meal suggestions (breakfast, lunch, dinner) based on users' Body Mass Index (BMI). The system uses K-Means clustering for user segmentation and Random Forest for accurate meal recommendations. I built the backend API using Flask and integrated data pipelines to preprocess nutritional datasets. The model achieved an accuracy rate of 87% in recommending optimal meals aligned with users' health goals.
Tools and Algorithms Used: Flask, Python, K-Means, Random Forest

Project 3

Visualizing Healthcare: An Analysis of Nursing Home Data

This project delivers an insightful analysis of U.S. nursing home data, focusing on key financial parameters like provider details, Medicare payments, and healthcare charges. Using D3.js for data visualization, the project features interactive scatter plots and bubble charts that illustrate the relationships between Medicare payments and covered charges, along with an exploration of average total payments. The visualizations allow users to interact with the data for clearer insights. The project leverages HTML, CSS, JavaScript, and the D3.js library to create dynamic, data-driven visualizations. Additionally, Bootstrap is used for responsive design, ensuring a seamless user experience across different devices. Data is sourced from CSV files, processed and visualized effectively to highlight critical trends in healthcare expenditure.
Tools & Technologies : D3.js, JavaScript, HTML, CSS, Bootstrap, CSV data.
